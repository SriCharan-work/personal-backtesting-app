**Personal Backtesting App**

This is a strictly local, non-commercial desktop application built with Electron and Node.js for my own personal trading education, manual backtesting and journaling.


**Purpose**

The goal of this project is to provide a free, local alternative to premium backtesting SaaS platforms. It is strictly non-commercial. It will never be hosted on a public web server, will never have a login system, will never be monetized, and will nto be distributed. It is an online desktop app solely by me for my personal use.


**Core Features (Planned)**

- Manual historical bar replay engine for Forex and NIFTY 50 Stocks.
- Interactive charting with drawing tools (trendlines, Fibs, etc.).
- Trade Execution simulation and local journaling (via SQLite).
- AI integration for personal trade analysis (using free-tier APIs).


**Data Sources**

- Forex Data: Fetched via open-source Dukascopy scrapers / free APIs.
- NIFTY 50 Data: Fetched via Yahoo Finance / personal broker APIs.


**Tech Stack**

- Electron / Node.js (Local Desktop App)
- TradingView Charting Library (Pending approval for non-commercial personal use)
- SQLite (for local trade journaling)
